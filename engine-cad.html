Robotic Arm Control System

Overview

This project implements real-time control of a 3-DOF robotic arm using inverse kinematics and Jacobian-based velocity control on embedded hardware.

System Design

  • Forward and inverse kinematics
  • Jacobian matrix and damped least squares
  • PID control with variable time step

Demo Video

What Went Wrong

Instability occurred near joint limits and Jacobian singularities, requiring damping and timing corrections.
